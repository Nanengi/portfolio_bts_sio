Annuaire téléphonique web

Description

Le projet d’annuaire téléphonique web est une application conçue pour gérer les contacts d’une organisation. Je ne l’ai pas créée : ma mission consistait à améliorer et mettre à jour un annuaire téléphonique existant. Cette application permet de consulter et de gérer les contacts d’une organisation. Il est également possible de mettre à jour les informations des contacts et d’assurer la maintenance de l’application afin de garantir son bon fonctionnement. Par ailleurs, elle donne accès à d’autres plateformes utilisées par l’entreprise.

Description

Dans ce cadre, j'ai été responsable de la conception backend et frontend de la page de gestion des comptes administrateurs, qui permet d'ajouter des gestionnaires sur la plateforme de réservation de salles.

Dreameez

Description

Dreameez est une application web de type réseau social permettant aux utilisateurs de créer un compte, de publier du contenu et d’interagir avec d’autres membres. Ce projet a été réalisé dans le cadre de ma première année de BTS SIO. J’ai été en charge du développement du backend de l’application, notamment la gestion des utilisateurs, des publications et des interactions entre les membres.
