Dreameez

Description

Dreameez est une application web de type réseau social permettant aux utilisateurs de créer un compte, de publier du contenu et d’interagir avec d’autres membres. Grâce à une intelligence artificielle intégrée, spécialement conçue pour cette application, il est possible de créer et de générer les rêves décrits par les utilisateurs. Ce projet a été réalisé dans le cadre de ma première année de BTS SIO.

Fonctionnalités

  • Création de compte / connexion
  • Publication de posts
  • Interaction (likes, commentaires)
  • Chat pour discuter avec d'autres utilisateurs

Technologies utilisées

HTML / CSS / JavaScript
